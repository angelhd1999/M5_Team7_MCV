## Use guide
1. Go to the desired week:
```
cd weekX :: Where X is the week number
```
2. (Optional) Create an environment.
3. Install the requirements at the folder:
```
pip install -r requirements.txt
```
4. Execute the run.py file:
```
python run.py
```
